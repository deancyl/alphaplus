## 一、项目概览

### 1.1 项目定位
像素级复刻专业投研工作台，构建零数据成本的个人投研系统，基于 FastAPI + Vue3 + SQLite + AkShare 构建。

### 1.2 核心特性
- **零数据成本**: 全部数据来自 AkShare 免费数据源
- **高性能查询**: SQLite WAL 模式 + Pandas 内存缓存，26,801只基金 ~7ms 响应
- **量化引擎**: 相空间轨迹计算、Pearson相关性、SLSQP因子暴露、Brinson归因
- **优雅降级**: GBM/O-U随机模拟器、Circuit Breaker熔断、指数退避重试
- **响应式设计**: 支持移动端320px到大屏3840px全尺寸适配

## 八、部署与运维

### 8.1 部署架构
- **后端**: FastAPI on port 60200
- **前端**: Vue3 on port 60201
- **数据库**: SQLite WAL模式
- **缓存**: Redis可选 (当前使用内存缓存)

### 8.2 环境要求
- Python 3.11+
- Node.js 18+
- npm 9+

### 8.3 启动方式
```bash
# 一键启动
./start.sh

# 或分别启动
python -m uvicorn backend.main:app --host 0.0.0.0 --port 60200
cd frontend && npm run dev
```

## 九、风险与挑战

### 9.1 外部依赖风险
- **AkShare API可用性**: 依赖第三方免费数据源
- **解决方案**: 实现了GBM/O-U模拟器降级机制

### 9.2 性能瓶颈
- **SQLite写入并发**: 单写入者限制
- **解决方案**: WAL模式 + 连接池 + 异步操作

### 9.3 架构限制
- **单机架构**: 无分布式支持
- **解决方案**: 适合个人使用，未来可扩展为微服务
